## 1. 박스 모델

- css로 그리는 모든 것은 박스다. 원으로 보이든 텍스트로 보이든 모든 것은 박스로 되어 있다.
- 박스 모델은 css의 핵심 기반이고 예측 가능한 css를 작성하는데 도움이 된다.

## 2. 콘텐츠 및 크기

- 박스의 크기는 다음 세가지 요소에 영향을 받는다.
  > 1. `display` 값
  > 2. 설정된 크기
  > 3. 박스 안 콘텐츠
- 박스는 `외부 크기 조정(extrinsic sizing)` 또는 `내부 크기 조정(intrinsic sizing)`을 사용하여 조정할 수 있다. 내부 크기 조정은 콘텐츠 기반으로 브라우저가 알아서 크기를 조정하게 둔다. 기본적으로 브라우저는 `내부 크기 조정`으로 가장 유연하게 박스 크기를 조정한다.

<Example.Example1 />

- 첫번째 박스: `width: 50px`로 고정되어 있다. 문제는 제한된 박스 크기보다 긴 콘텐츠가 있다는 점이다. 해당 콘텐츠는 부모의 border box 밖으로 overflow된다.
- 두번째 박스: 첫번째 박스처럼 부모 밖으로 넘어가는 것을 방지하기 위해서는 두번째 예시처럼 `width`를 설정하지 않거나
- 세번째 박스: 세번째 박스처럼 `width: min-content`로 설정하는 방법이 있다. `min-content`는 콘텐츠의 최소 내부 너비(단어 "intrinsic")만큼 넓어지도록 한다.

## 3. 박스 모델 영역

![box-model-area](https://web.dev/static/learn/css/box-model/image/three-photo-frames-411b8e31c2b93_1920.jpg?hl=ko)

- content box: 콘텐츠 영역이다. 콘텐츠는 부모의 사이즈를 결정한다.
- padding box: `padding` 속성에 의해 생성된 영역이다. 박스 경계와 콘텐츠 사이에 간격을 주고 싶을 때 `padding` 값을 설정한다. 스크롤바도 padding box 내에 생성된다.
- border box: `border` 속성에 의해 생성된 영역이다. 박스의 시각적 경계이다.
- margin box: `margin` 속성에 의해 생성된 영역이다. 주변 요소 간에 간격을 주고싶을 때 `margin` 값을 설정한다. `outline` 및 `box-shadow`가 해당 영역을 차지하는데 박스 크기에 영향을 미치지 않는다.

## 4. 박스 모델 제어하기

- 모든 브라우저는 HTML에 사용자 에이전트 스타일시트를 적용한다. css가 정의되지 않았을 때 적당한 가독성을 위해 기본값을 제공해준다. 예를 들어, `<div>`의 기본 `display`값은 `block`이고 `<span>`의 기본 `display`값은 `inline`이다.
- 기본적으로 모든 요소에는 `box-sizing: content-box;` 사용자 에이전트 스타일시트가 적용된다. `box-sizing` 속성은 박스 모델 크기 계산 방법을 정하는 속성이다. `width`, `height` 속성에 영향을 준다.

<Example.Example2 />

- 둘다 `width: 200px;`이고 동일한 `padding`, `border` 값을 주었는데 사이즈가 다르다.
- `box-sizing: content-box;`: content box의 `width`가 200px이다. box-sizing의 기준이 content box이다.

```text showLineNumbers
// 전체 박스 크기
200px(content box) + 20px*2(padding) + 10px*2(border) = 260px
```

- `box-sizing: border-box;`: border box까지의 `width`(content+padding+border)가 200px이다. box-sizing의 기준이 border box이다.

```text showLineNumbers
// 전체 박스 크기 200px 고정, content box 크기
200px - 20px*2(padding) - 10px*2(border) = 140px
```

- 요소에 `box-sizing: border-box;`를 적용하면 박스 모델 사이즈가 더 예측 가능하기 때문에 모든 요소에 기본값을 global하게 바꾸는 경우도 있다.

```css showLineNumbers
*,
*::before,
*::after {
  box-sizing: border-box;
}
```
